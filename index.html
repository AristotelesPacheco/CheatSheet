CSS Selectors and Specificity

Selectors

Selector Name Description
element_name (p, h1, a, ...) Type The Type selector matches the type of the element in the HTML document.
* Universal The universal selector (*) selects all elements of any type.
.class_name Class To select an HTML element by its class using CSS, a period (.) must be prepended to the class’s name.
class_name1,class_name2,etc. Multiple Classes We can add multiple classes to an HTML element’s class attribute by separating them with a space.
#id_name ID To select an element’s ID with CSS, we prepend the id name with a number sign (#).
[Attribute_name] Attribute The attribute selector can be used to target HTML elements that already contain attributes. The most basic syntax is an attribute surrounded by square brackets.
selector_name:pseudo_class_name Pseudo Class A pseudo-class can be attached to any selector. It is always written as a colon : followed by a name. For example p:hover.
